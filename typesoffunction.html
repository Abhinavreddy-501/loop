Arrow Function

Arrow function is a concise way of writing the function and it is introduce in ecma script

An anonymoous function in javascript is a function without a name. It is often used in situations where a function is needed temporarily, such as passing a function as an argument to another function

In javascript, variables can have either global or local scope. Global scope means the variable is accessible throughout the entire program, while local scope means the variable is only accessible with in a specific block or function.

A closure is the combination of a function bundled together (enclosed) with references to its surrounding state (the lexical environment). In other words, a closure gives you access to an outer function's scope from an inner function.

In javascript, a pure is a function that, given the same input, will always returns the same output and has no side affects(does not modify the state)

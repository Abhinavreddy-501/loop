While loop

In javascript, a while loop is a control flow statement that allows you to repeatedly execute a block of code as long as a specified condition is true.

Do while loop

In javascript, a do-while loop is a control flow of statement that repeatedly execute a block of code as long as a specified condition evaluates to true

for loop

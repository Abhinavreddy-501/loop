There are 3 condition checking operators

  1. If condition
  2. if-else
  3. Switch condition

If condition

The if statement is most basic conditional statement. it evaluates a condition and,if the condition is true, it executes a block of code

If-else

The if-else statement allows you to execute a one block of code if the condition is true, and another block if the condition is false

Ternary operator

The ternary operator is a concise way to write an if-else statement in a single line.

Switch Statement

In javascript, the switch statement is a control flow statement that allows a program to evaluate an expression against multiple possible case values. It provides a more concise way to write multiple if-else statements when you want to compare a single value against several possible values.
